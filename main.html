Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are getElementById(), getElementsByClassName(), querySelector(), or querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference are innerHTML gets HTML content, and textContent returns all text including hidden elements.

Question-3: What is event delegation in the DOM?

It’s when you attach a single event listener to a parent element to handle events for its child elements.

Question-4: What is event bubbling in the DOM?

It’s when an event starts at the target element and propagates up through its ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements using JavaScript are createElement() to create, appendChild() to add, and removeChild() to remove elements.
